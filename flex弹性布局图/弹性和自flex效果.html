Web的Flex弹性盒模型

[ flex ]子元素 该属性是flex-grow,flex-shrink和flex-basis的缩写，默认值为 0 1 auto。后两个属性可选

建议优先使用这个属性，而不是单独写三个分离的属性，因为浏览器会推算相关值

display: -webkit-flex;

width:100px;
flex: 2;
flex: 1;

display: -webkit-flex;

width:150px; margin-right:10px;
flex: 1;
width:150px; margin-left:10px;

[ flex-flow ] 复合写法，可以有两个属性: flex-direction flex-wrap; /* 自个体会吧 */

[ justify-content ]给父级 规定弹性盒伸缩子元素在伸缩容器内的对齐方式

/* 效果如同 float:left; */

[ align-self ]子元素 允许单个子元素有与其他子元素不一样的对齐方式，可覆盖align-items属性， 可以继承父元素的align-items属性，如果没有父元素，则等同于stretch

display: -webkit-flex;

align-self:flex-start;
align-self:flex-end;
align-self:center;
align-self:auto; /*default */
align-self:baseline;
align-self:stretch;

[ align-items ]给父级 规定弹性盒交叉轴的对齐方式

align-items: flex-start; /* default */

display: -webkit-flex;

width:100px;
width:100px;

align-items: flex-end;

display: -webkit-flex;

width:100px;
width:100px;

/* 第三代垂直居中 */

align-items: center;

display: -webkit-flex;

width:100px;
width:100px;

注意：根据文字基线对齐

align-items: baseline;

display: -webkit-flex;

width:100px;
width:100px;

align-items: stretch;

display: -webkit-flex;

height:100px;
min-height:20px;
max-height:60px;
